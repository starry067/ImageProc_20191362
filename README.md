# C-based Image Processing Practice (ImageProc_20191362)

이 프로젝트는 학부 과정에서 수행한 **C 언어 기반 기초 영상처리 실습 과제**입니다.  
BMP 이미지 파일을 직접 읽고, 픽셀 단위로 연산하며 다양한 영상 필터를 구현하는 것을 목표로 했습니다.

로우레벨 이미지 처리 과정을 직접 구현해보며  
**2차원 배열, 포인터, 메모리 구조, 픽셀 연산, 필터링 알고리즘 원리**를 경험할 수 있었던 프로젝트입니다.

---

## 📆 Project Period (진행 기간)

- **2021년 9월 ~ 2023년 12월 20일**  
  2년간의 영상 처리 실습 및 응용 과제로 진행되었으며,  
  기초 알고리즘 학습부터 다양한 필터 구현까지 순차적으로 수행하였습니다.

---

## 📌 Implemented Features (구현 기능)

### 🔹 기본 처리
- Grayscale 변환  
- 이미지 반전(Negative)  
- 밝기 조절(Brightness)  
- 임계값 처리(Thresholding)  

### 🔹 필터링 / 컨볼루션
- Averaging Filter (블러)  
- Sharpening Filter  
- Edge Detection (Sobel, Laplacian 등)  

### 🔹 파일 입출력
- BMP 파일 헤더 분석  
- Raw pixel 데이터 읽기/쓰기  
- 24bit BMP 포맷 처리
